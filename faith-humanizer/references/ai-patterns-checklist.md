# AI写作指纹检测清单 v2.3 — 四声部校准版

> **用途**: faith-humanizer Pass 1-3 的模式匹配参考。
> **v2.3 变化: 新增M类（AI概率最大化句式）——来自v1→v3三轮迭代的AI指纹提取。核心发现：AI用"省力连接"替代逻辑衔接。四声部校准：dontbesilent + 认知觉醒 + 被讨厌的勇气 + 专注的艺术。
> 正向参考见 [`positive-writing-patterns.md`](positive-writing-patterns.md)（v2.3 四声部）。

## 优先级定义
- 🔴 高优先级（第一轮处理）：A类固定句式 + B类虚假深度 + G类真实性缺陷 + **L类AI口播低级伪造（v2.2 NEW）**
- 🟡 中优先级（第二轮处理）：C类排版脚手架 + D类空话泛话 + H类结构臭虫
- 🟢 可延后：E类引注格式
- 🔵 全程：F类标题标点 + I类灵魂缺失 + J类人类特征缺失 + K类认知觉醒风格伪造

## A. 固定句式（🔴 高优先级）
| # | 模式 | AI指纹识别 | 改写方向 |
|---|------|-----------|---------|
| A1 | 三段论引导："首先…其次…最后" | 模板化编号词 | 用"第一…第二…"替代，或去掉编号自然递进 |
| A2 | 模板转折："值得注意的是""不可否认的是" | 无信息量的路标词 | 删掉路标，直接陈述 |
| A3 | 负平行结构："不仅仅是X，更是Y" | 先否定再肯定的模板 | 只说Y。X如果无关就不提 |
| A4 | 路标宣告："让我们""下面我们来看" | 教学腔，非对话 | 直接说内容 |
| A5 | 说服权威腔："真正的问题是""本质上" | 假装深刻的空话 | 删除修饰，直接说观点 |
| A6 | 挑战+希望模板："尽管面临X但Y充满希望" | 政论文结尾模板 | 替换为具体事实或删除 |
| A7 | 开放式结尾："未来充满机遇""前景广阔" | 万能废话结尾 | 替换为具体下一步或直接删除 |
| A8 | 知识截断声明："据目前可获得的资料" | AI的免责话术 | 直接陈述已知信息 |
| A9 | chat式交互残留："希望有帮助""当然，" | 客服腔 | 删除，正常陈述 |
| A10 | 正面过热腔："非常好的问题，" | 谄媚式回应 | 正常回应或不回应 |

## B. 虚假深度（🔴 高优先级）
| # | 模式 | 改写方向 |
|---|------|---------|
| B1 | 空泛AI高频词："至关重要的""深刻的""重大的" | 换成具体描述或删除 |
| B2 | 促销宣传腔：boasts/exemplifies/nestled/highlighting | 换简单动词 |
| B3 | 伪比喻打引号："土壤""天花板""底层逻辑" | 去掉引号，正常叙述 |
| B4 | 虚假范围："从X到Y，从A到B"跨维度排比 | 拆成独立陈述 |
| B5 | 空洞填充：highlighting/reflecting/showcasing | 换具体动作或删除 |
| B6 | 笼统归属："专家认为""研究表明"无来源 | 给具体来源或删除 |
| B7 | 省略系词：serves as/stands as 代替 is | 直接用"是" |
| B8 | 同义替换强迫症——同一个概念换3+种说法 | 统一用同一个词 |
| B9 | 夸张结论："重大突破""前景广阔" | 给数据或删除 |
| B10 | 问句开头装互动："你是否曾想过" | 直接陈述 |

## C. 排版脚手架（🟡 中优先级）
| # | 模式 | 改写方向 |
|---|------|---------|
| C1 | 连续 3+ 行 <40 字的短行（非口语爆破，是列表式短行） | 合并为复合句 |
| C2 | bullet/numbered list 占比 >30% | 溶解为连续性叙事 |
| C3 | "**标题**: 描述"重复 3+ 次 | 改为段落叙述 |
| C4 | 连续 3+ 段每段正好 3-4 句 | 打乱长短节奏 |
| C5 | 每段首句永远是主题句 | 自然过渡，允许首句是细节 |
| C6 | emoji 符号装饰 | 删除 |
| C7 | 频繁用"——"代替逗号句号 | 正常使用标点 |

## D. 空话泛话（🟡 中优先级）
| # | 模式 | 改写方向 |
|---|------|---------|
| D1 | 填充短语：in order to/due to the fact | 换短版本 |
| D2 | 可能/也许/或许/在某种程度上连锁出现 | 留一个或全部删除，给明确判断 |
| D3 | 抽象名词堆砌：赋能/闭环/底层逻辑/抓手/认知升级 | 拆成具体描述 |
| D4 | 被动/无主语句："需要更多的研究" | 给明确主语 |
| D5 | 三段空骨架：论点→论据→结论机械模板 | 打破推进顺序 |

## G. 真实性缺陷（🔴 高优先级——红线）
| # | 模式 | 改写方向 |
|---|------|---------|
| G1 | 编造个人经历故事 | 删除或换真实案例 |
| G2 | 虚构细节增感染力 | 不添加虚假细节 |
| G3 | 以"我"开头假装亲历 | 不确定则用第三人称 |
| G4 | 假借名家之口编故事 | 必须有可靠出处 |
| G5 | 模糊时间地点 + 第一人称 | 给真实细节或删除 |
| G6 | 借第三方虚构："我认识一个人" | 用真实案例替代 |
| G7 | AI式口语模仿："咱们都知道""你想啊"——假装对话但无真实互动感 | 删除，或替换为真实的具体场景 |

## H. 结构臭虫（🟡 中优先级）
| # | 模式 | 改写方向 |
|---|------|---------|
| H1 | 三段式强行分组 | 拆成两段或一段 |
| H2 | 同义替换强迫症 | 恢复同一个词 |
| H3 | 标题后暖场废话 | 直接进内容 |
| H4 | 虚假排比范围 | 拆开陈述 |

## I. 灵魂缺失（🟡 中优先级）
| # | 模式 | 改写方向 |
|---|------|---------|
| I1 | 每句等长等结构 | 长短句交替，允许≤5字的爆破句 |
| I2 | 全是陈述没有观点 | 加入个人判断和情绪 |
| I3 | 读起来像操作手册 | 去掉步骤语言，注入叙事 |

| I6 | 无人称锚定——全文没有"你"字 | 加入至少3处"你"字直呼 |

## J. 人类特征缺失（🔵 全程）

> **核心原理**: AI文本的问题不仅是包含错误模式，更是**缺少**人类写作的结构性特征。

| # | 检测项 | 阈值 | 不达标处理 |
|---|--------|------|-----------|
| J1 | 口语流标记密度 | ≥1个/200字 | 在自然位置嵌入 |
| J2 | 动词重叠式 | ≥1个/500字 | 将普通动词改为重叠式 |
| J3 | 短句爆破比例 | 短句(≤10字)占比 ≥30%（但注意L2：不能一句一段） | 拆分长句，插入短判断句，但保持段内排列 |
| J4 | "第一…第二…"替代"首先其次" | 如有列举，100%使用口语编号 | 替换模板化编号 |
| J5 | 直接判断句密度 | ≥1个/300字 | 去掉软化词 |
| J6 | 具体场景锚定 | 每个抽象概念后 ≥1个"比如说…"场景 | 补充具体例子 |
| J7 | 反总结结尾 | 结尾无"综上所述/总而言之" | 改为具体收尾 |
| J8 | 口语词汇占比 | ≥70%动词来自HSK 1-4级 | 替换书面化动词 |
| J9 | 自问自答 | 开头或转折处 ≥1处自问自答（但注意L7：必须展开回答≥3句） | 设问后充分回答 |
| J10 | 推理过程外显 | 关键论点使用"问→假设→检验→结论"链条 | 展开推理步骤 |
| J11 | 第二人称直呼 | "你"字 ≥3次/全文 | 将泛指改为"你"直呼 |
| J12 | 情绪表达 | ≥1处情绪词，但注意L8：必须有场景支撑 | 情绪词前给触发场景 |

## K. 认知觉醒/周岭风格伪造（🔵 全程）

> **核心原理**: AI尝试模仿书面叙事风时的典型翻车模式。

| # | 模式 | AI伪造特征 | 改写方向 |
|---|------|-----------|---------|
| K1 | 假"我们"共情腔 | "我们应该""我们必须"——说教套了共情的壳 | 改真共情或退回"你"直呼 |
| K2 | 空洞故事开场 | 无细节、无场景、无人物的小寓言 | 用真实案例或直接陈述 |
| K3 | 隐喻堆砌不发展 | 一段抛3+个比喻但每个只存在一句 | 保留1个，至少3句话展开 |
| K4 | 名人名字掉书袋 | 堆砌名字不阐释观点 | 保留1位+具体观点 |
| K5 | 伪共识宣告 | "众所周知""我们都知道" | 替换为第一人称判断 |
| K6 | 伪读者互动 | "一位读者告诉我…"无具体细节 | 有真实来源才写 |
| K7 | 科学数据空洞引用 | "研究表明"无数据/来源 | 给具体数字+来源或删除 |
| K8 | 流水账式递进 | "此外""另外""还有一点" | 改用因果/转折/递进 |
| K9 | 温暖空洞结尾 | "愿我们都能""让我们一起" | 给行动建议或回扣开篇 |
| K10 | 概念创造强迫 | 给普通名词加引号当概念，不复用 | 只创造会被复用的概念 |
| K11 | 数据比喻机械拼接 | 数据→比喻中间无自己的解读 | 先解释数据含义，再自然过渡 |
| K12 | 章节感过强 | 小标题/分隔线切碎文本 | 用过渡句串联 |

## M. AI概率最大化句式（🔴 高优先级——v2.3 NEW）

> **核心发现（来自v1→v3三轮迭代）**: AI文本的深层指纹不是某个具体的词，是一种**省力连接策略**。AI用破折号替代句号（跳过句子间逻辑衔接）、用"大多数人"替代举证（跳过具体描述）、用"一句话概括"替代精炼表达（跳过真正的简化工作）。真实书籍作者相反——每一句和下一句的衔接靠因果、转折、递进逻辑自然推动，不靠标点符号跳过去。

| # | 模式 | AI省力策略 | 检测阈值 | 改写方向 |
|---|------|-----------|---------|---------|
| M1 | `——` 破折号万能连接器 | 用破折号替代句号/逗号，跳过句子间的逻辑衔接工作。真实中文书籍破折号密度约1/800-1000字；AI文本可达1/100字 | >1个/300字 = FAIL | 每处破折号改回句号/逗号/冒号。保留仅用于合法的成对插入语（如"那个决定——多睡二十分钟还是多刷二十分钟手机——带来的代价"） |
| M2 | "大多数人/很多人/无数人" 统计腔 | 用模糊统计词替代具体描述或举证。真实作者要么给数字，要么不说"大多数" | >0次/全文 = 审查 | 删除"大多数人"。直接描述现象。如果确需统计数据，给出具体数字和来源 |
| M3 | "一句话概括/简单来说/归结为" 降维腔 | 作者假装替读者简化，实际是跳过精炼表达的工作。真实作者一次说清楚，不需要二遍简化 | >0次/全文 = 审查 | 删除降维宣告。把后面的内容直接写清楚——如果你能"一句话概括"，那一句话本身就是正文 |
| M4 | "X吗？不是/是的。" 自问自答腔 | AI用假对话制造互动感。真实写作者极少在正文中自问"X吗"然后自答 | >0次/全文 = 审查 | 删除问答对。直接陈述。如"多巴胺受体的下调是不可逆的吗——不是"→"多巴胺受体的下调是可逆的" |
| M5 | "当...的时候" N连排比段首 | 连续2+段以"当...的时候"开头。AI的模板化平行结构，真实作者会变换段首句式 | 连续2+段 = FAIL | 变换段首。用"如果...""一旦...""在...状态下"等不同句式替代 |
| M6 | "你可能会想/你会发现/你会注意到" 预判腔 | AI替读者预判反应。真实作者描述发生了什么，不替读者预测"你可能会想什么" | >0次/全文 = 审查 | 删除预判句。直接描述现象或结论，让读者自己去"发现" |
| M7 | "这就是为什么/这就是" 总结腔 | AI统一使用"这就是"作为所有结论的引导词。真实作者用"所以/因此/于是/由此/可见"等多种信号 | "这就是" ≠ 自动FAIL | 非绝对禁止，但如果全文所有结论都用"这就是"引导，替换为多样化过渡词 |
| M8 | "X的本质是Y/X的核心是Y" 定义腔 | AI对被讨论的概念做不必要的重新定义。真实作者在引入概念的同时就定义好了，不需要后补"X的本质是..." | >0次/全文 = 审查 | 删除定义宣告。如果概念需要定义，在首次出现时自然带出，不单独起一句"X的本质是..." |

**M类机制原理**: 以上8项的共性不是"某个词不能用"，是**AI用形式替代了实质**。破折号替代了句子间逻辑设计。"大多数人"替代了举证。"一句话概括"替代了精炼表达。反AI的核心不是禁用这些词，是强迫自己完成它们替你跳过的那一步工作。

## L. AI口播风低级伪造（🔴 高优先级）

> **核心原理（v2.3深化）**: AI在看到"口播风""短句""口语化"指令后，会产生一套特定的低级伪造模式。这些不是风格问题——它们是**逻辑错误**。v2.3新增每条L项的机制解释：说明"为什么AI会生成这个"以及"为什么真实人类不这么写"。

| # | 模式 | AI伪造特征 | 改写方向 |
|---|------|-----------|---------|
| L1 | "不是...是..."虚假对立 | "不是不努力。是搞错了顺序。"——AI用两句对仗替代了整段因果推理。"为什么是B而不是A"的论证被压缩成了两个句号 | 展开为推理过程："以前以为是X→后来发现不对→因为出现了Y证据→所以实际是Z"。用因果链替代对仗 |
| L2 | 一句一段（伪短句爆破） | 每个句号后换行——AI用视觉分段伪装节奏感。真实口语的节奏在句长交替，不在排版。AI这样做既能省token又能假装"有节奏" | 合并为长短交替的段落。短句(≤10字)段内存在，不独占一行。单句段占比≤15% |
| L3 | 不合逻辑的夸张/比喻 | "那种糖比菌多"——AI在不同领域词汇间随机采样拼贴比喻，不验证本体和喻体是否在同一比较维度。菌落数和含糖量不属于同一计量范畴 | 删除。替换为同维度对比，或给出实际数据。如果比喻的本体和喻体不能放在同一个句子里被"比"字连接而逻辑自洽，就是L3 |
| L4 | 自言自语/元评论 | "你看，我讲了这么多"——AI对自己正在写作的行为进行元评论。真实作者不评论自己的写作过程，他们让内容自己说话 | 全部删除。用内容逻辑衔接下一段 |
| L5 | 伪口语粒子过密 | "对吧""你看""那""就"密度 >1个/100字，且分布不均 | 控制在1个/150-200字，均匀分布在转折处 |
| L6 | 信息量为零的短判断 | "真的。""就这么简单。"——AI用句号制造"爆破感"，但句号后面的内容没有任何新信息 | 删除或合并。短句必须携带新信息 |
| L7 | 设问敷衍答 | 抛出问题后用一句话草草带过 | 设问后至少展开3句话回答，或删除设问 |
| L8 | 情绪词无场景 | 情绪词连续出现但无触发场景 | 每个情绪词前≥1句场景描述 |
| L9 | 废话重复强调 | "说真的""讲真的"——无新增信息的重复 | 删除。重要则从新角度展开 |
| L10 | 伪造现场感 | "我刚看到一个..."——时间戳假装即兴 | 给真实时间+事件或删除标签 |
